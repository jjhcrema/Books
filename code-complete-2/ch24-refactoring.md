## 24.1 Kinds of Software Evolution
- 프로그램의 품질이 떨어지고 있는 것을 발견한다면, 프로그램이 잘못된 방향으로 진화하고 있다는 것

## 24.2 Introduction to Refactoring
- 소프트웨어를 보다 쉽게 이해할 수 있고, 적은 비용으로 수정할 수 있도록 겉으로 보이는 동작의 변화 없이 내부 구조를 변경한 것
